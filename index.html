PopulationRank

My App lists the top 40 countries ranked by population (most populous to least)

  • It takes in 1 input, in the form of a Slider
  • Moving the slider will allow the User to select a country by Rank
  • A CSV with the names, ranks of the Countries (sorted on rank) is read in
  • This input (rank) is then processed to fetch the Country Name and Map
  • The Country Name and Map are displayed

Code for Rendering Maps

The .csv containing ranks and names of countries is read in and used as input to map function.

fileName <- "C:/Users/MALPA06/Documents/R Workspace/Module_9/Project/population/Countries_And_Population.csv"
